# ADHD-HIV-PrEP website
Source code for the ADHD-HIV-PrEP website — a Quarto site (in English, French and German) presenting our work on the intersection of neurodiversity (ADHD) and HIV, including people living with HIV and people taking HIV pre-exposure prophylaxis (PrEP), starting with the ADHD-HIV-PrEP survey.

Many thanks to Christof Schönenberger for generously sharing the website template from the [RETUNE trial](https://github.com/christofmanuel/RETUNE_website), which this site was built on.
